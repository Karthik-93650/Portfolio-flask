# Flask Portfolio Website

## Description
A personal portfolio website built using **Flask** to showcase profile, skills, and projects.  
This project demonstrates Flask routing, HTML templates, static files, and basic deployment setup.

---

## Features
- Home page with profile details
- About Me section
- Skills list
- Projects list
- Contact page
- Clean UI using HTML & CSS
- Flask backend

---

## Tech Stack
- Python
- Flask
- HTML
- CSS
